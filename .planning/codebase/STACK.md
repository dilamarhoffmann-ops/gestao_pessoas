# Stack

## Languages
- TypeScript (frontend and backend)
- HTML/CSS (via Tailwind CSS)

## Runtime & Build
- Node.js (via `tsx` for backend execution)
- Vite (bundler and dev server)
- pwsh (PowerShell scripts used for tooling like `dev.ps1`, `sync.ps1`)

## Frameworks & Libraries
- **Frontend**: React 19, React Router DOM, Lucide React (icons), TailwindCSS v4, Framer Motion (animations)
- **Backend**: Express.js
- **Database**: SQLite (via `better-sqlite3`), Supabase (via `@supabase/supabase-js`)
- **Other**: OpenAI API, Google GenAI SDK, AWS S3 Client, Multer (file uploads), PDF Parse

## Configuration
- `package.json` (npm scripts and dependencies)
- `vite.config.ts` (Vite config, proxies `/api` to localhost:3000)
- `tsconfig.json` (TypeScript config)
- `.env` (environment variables)
